What you must do after download:

4. Upload this folder to GitHub and deploy on Vercel (root should contain index.html).

Files included:

- admin.html  (admin login + add products)
- style.css   (site styling)
- firebase.js (paste your Firebase config here)
- script.js   (site logic)
- images/     (empty; placeholder.png included)
